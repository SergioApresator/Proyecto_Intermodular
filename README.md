# EPICKEYS
Tecnologías usadas: Spring boot, Angular, VSCode, MySQL, XAMPP.

Instrucciones de instalación: 
1 - Clonar repositorio.
2 - Añadir el backend en Spring Boot.
3 - Añadir frontend en VSCode.
4 - Encender la base de datos mediante XAMPP.
5 - Arrancar el backend.
6 - Entrar en la dirección del frontend.
